# 《Lose in Head》3D 叙事解谜原型

## 项目概述

《Lose in Head》是一款以迷宫探索和线索拼合为核心的 3D 叙事解谜原型。玩家通过探索场景、开启宝箱、收集人物档案与记忆碎片，逐步进入光路谜题阶段并揭示事件真相。

## 核心玩法循环

`迷宫探索 → 宝箱交互 → 收集并回看线索 → 激活光路谜题 → 解锁真相回响 → 开启出口`

## 主要系统

- 迷宫探索、第三人称移动与场景交互；
- 14 组宝箱、人物线索与记忆碎片；
- 线索背包与信息回看；
- 镜面、光束、机关墙和谜题接收器；
- 序章、阶段切换、真相回响与结局流程；
- 主菜单、操作提示和退出流程。

## 项目重点

项目重点验证探索、叙事内容和谜题目标如何形成连续流程，并通过阶段反馈让玩家理解当前目标与故事进展。
