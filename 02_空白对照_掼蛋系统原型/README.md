# 《空白对照》多人掼蛋系统原型

## 项目概述

项目以传统掼蛋规则为基础，对牌型、逢人配、炸弹比较、轮转、完牌名次、升级和进贡还贡等内容进行结构化整理，并尝试扩展为四人、六人和八人模式。

## 设计目标

- 将存在地区差异的规则整理为明确、可验证的规则表；
- 把玩家人数、牌副数、队伍人数与结算关系抽象为配置；
- 使用统一逻辑处理不同人数模式的升级与名次结算；
- 为机器人、状态同步和后续联机功能预留模块接口。

## 模块结构

- Cards：卡牌与牌堆；
- Rules：牌型识别和大小比较；
- GameState：对局状态；
- TurnFlow：回合与过牌流程；
- PlayerSeat：座位和队伍；
- Bot：托管机器人；
- Serialization：状态序列化；
- Networking：联机接口；
- UI：主菜单、大厅和牌桌表现。

## 验证方式

通过 EditMode 测试验证发牌、牌型识别、牌型比较、座位、回合、结算和状态转换等逻辑。
