# 《大耳帽兜桌面宠物》轻量互动原型

## 项目概述

使用 Unity 制作的 Windows 桌面宠物原型，通过透明无边框窗口让角色显示在桌面上，并提供轻量的自主行为和鼠标互动。

## 主要功能

- 透明桌面窗口；
- 待机、左右奔跑、跳跃和挥手动画；
- 一定范围内的随机等待与漫游；
- 鼠标拖动角色；
- 点击与松开时的动画反馈。

## 设计重点

使用随机等待、方向变化和动画反馈增强角色的生命感，同时保持操作简单，不干扰用户正常使用桌面。
