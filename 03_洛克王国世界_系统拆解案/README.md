# 《洛克王国：世界》系统拆解案

## 拆解目标

从系统策划视角分析游戏各模块的定位、相互关系与玩家目标，梳理核心循环、资源闭环、养成层级和长线内容结构。

## 拆解范围

- 战斗系统；
- 精灵获取与精灵养成；
- 开放世界探索；
- 角色养成；
- 资源系统；
- 赛季系统；
- 家园系统；
- 社交与周边系统。

## 核心循环

`开放世界探索 → 获取精灵与资源 → 精灵养成 → PVE/PVP 战斗 → 获得高阶资源与新目标 → 返回探索`

## 分析重点

- 探索、收集、养成和战斗如何形成闭环；
- 不同养成层级对应的玩家分层；
- 赛季资源与长期目标如何控制内容消耗；
- 家园和社交系统如何提供情感价值与留存动力；
- 策略深度、重复养成和后期驱动力等问题及优化方向。
