# 范自坤（erhuiui）｜游戏系统策划作品集

## 个人简介

我叫范自坤，目前是中国科学院大学资源与环境专业研二硕士研究生。虽然我的专业不是游戏方向，但研究生阶段长期做数据分析、模型构建等相关工作，培养了较强的逻辑思维、规则拆解和文档表达能力。我认为这些能力和游戏策划的岗位要求是比较契合的。

## 求职方向

- 游戏执行策划
- 游戏系统策划
- 游戏关卡策划
- 游戏数值策划

## 作品目录

### 01. 《洛克王国：世界》系统拆解案

[查看项目说明](./01_洛克王国世界_系统拆解案/README.md) ｜ [查看 XMind 拆解文件](./01_洛克王国世界_系统拆解案/洛克王国.xmind)

从战斗、精灵获取与养成、开放世界探索、资源、赛季、家园和社交等维度分析系统定位与关联，梳理“探索—捕捉—养成—战斗”核心循环，并提出问题判断与优化方向。

### 02. 《Lose in Head》3D 叙事解谜原型

[查看项目说明](./02_Lose_in_Head/README.md) ｜ [查看实机视频](./02_Lose_in_Head/实机视频.mp4) ｜ [下载试玩版](https://github.com/erhuiganlanshi/game-design-portfolio/releases/tag/lose-in-head-playtest-v1)

玩家在迷宫中探索人物线索，通过宝箱、镜面、光路和机关谜题逐步拼合事件真相。项目重点展示探索、叙事、线索收集与谜题系统之间的流程衔接，以及策划方案在 Unity 中的原型落地。

**关键词：** 叙事解谜、探索循环、线索系统、关卡流程、Unity 原型

### 03. 多人掼蛋扩展系统原型（开发中）

[查看项目目录](./03_多人掼蛋扩展（开发中）/) ｜ [规则整理](./03_多人掼蛋扩展（开发中）/掼蛋规则整理.md) ｜ [开发计划](./03_多人掼蛋扩展（开发中）/在线掼蛋第一版开发计划.md) ｜ [模块化构建计划](./03_多人掼蛋扩展（开发中）/模块化构建计划.md)

以传统掼蛋规则为基础，整理牌型、逢人配、炸弹、回合、升级和进贡还贡等系统，并尝试将四人玩法扩展为四人、六人和八人可配置模式。当前重点是规则内核拆分、多人模式泛化、好友房联机方案与模块化开发顺序。

**关键词：** 规则设计、系统泛化、状态管理、回合流程、联机方案、测试验证

### 04. Erhui 桌面宠物轻量互动原型（开发中）

[查看项目说明](./04_Erhui_桌面宠物（开发中）/README.md) ｜ [GDD](./04_Erhui_桌面宠物（开发中）/01_GDD.md) ｜ [TDD](./04_Erhui_桌面宠物（开发中）/02_TDD.md) ｜ [数据结构](./04_Erhui_桌面宠物（开发中）/03_DataStructure.md) ｜ [行为逻辑](./04_Erhui_桌面宠物（开发中）/04_Behavior.md)

使用 Unity 制作的 Windows 桌面宠物原型，通过透明无边框窗口让角色显示在桌面上，并实现随机等待与漫游、动画状态切换、鼠标拖动和点击反馈。

**关键词：** 桌面宠物、交互反馈、行为状态、动画切换、Unity
