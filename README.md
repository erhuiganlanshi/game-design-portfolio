## 作品目录

### 01. 《Lose in Head》3D 叙事解谜原型

[查看项目说明](./01_Lose_in_Head/README.md)

玩家在迷宫中探索人物线索，通过宝箱、镜面、光路和机关谜题逐步拼合事件真相。项目重点展示探索、叙事、线索收集与谜题系统之间的流程衔接，以及策划方案在 Unity 中的原型落地。

**关键词：** 叙事解谜、探索循环、线索系统、关卡流程、Unity 原型

---

### 02. 《空白对照》多人掼蛋系统原型

[查看项目说明](./02_空白对照_掼蛋系统原型/README.md)

以传统掼蛋规则为基础，整理牌型、逢人配、炸弹、回合、升级和进贡还贡等系统，并尝试将四人玩法扩展为四人、六人和八人可配置模式。

**关键词：** 规则设计、系统泛化、状态管理、回合流程、测试验证

---

### 03. 《洛克王国：世界》系统拆解案

[查看项目说明](./03_洛克王国世界_系统拆解案/README.md)

从战斗、精灵获取与养成、开放世界探索、资源、赛季、家园和社交等维度分析系统定位与关联，梳理“探索—捕捉—养成—战斗”核心循环，并提出问题判断与优化方向。

**关键词：** 系统拆解、核心循环、资源闭环、长线留存、优化分析

---

### 04. 《大耳帽兜桌面宠物》轻量互动原型

[查看项目说明](./04_大耳帽兜_桌面宠物/README.md)

使用 Unity 制作的 Windows 桌面宠物原型，实现透明窗口、随机漫游、动画状态切换，以及鼠标拖动和点击反馈。

**关键词：** 桌面宠物、交互反馈、行为状态、动画切换、Unity
